# elf4js

ELF/Dwarf parser for Java Script.

## Features

- Elf headers
- ELF section structure
- String Table
- String Offset Table
- Symbol Table
- Address Table
- Abbreviation Table
- Line Program Information
- (Partial) Debug Information

## Limitations

Debug Information section is just partially implemented and therefore disabled by default.
